# 삼표 특수몰탈 위키 - 사용 가이드

> Production: https://vapmortarwiki.vercel.app

## 1. 주요 메뉴

| 메뉴 | 기능 |
|------|------|
| **대시보드** | 전체 통계, 최근 문서, AI 배너 |
| **제품 DB** | 삼표 특수몰탈 15종 (그라우트/보수/바닥/타일) |
| **위키 문서** | 기술 해설, 시장 동향, 경쟁사 분석 16건 |
| **PDF 문서** | 업로드된 PDF 목록 (경쟁사 TDS/가이드) |
| **AI 질의응답** | 자연어 질문 → AI 답변 (제품 물성, 시공법, 규격 등) |
| **통합 검색** | 제품/위키/PDF/규격 통합 검색 |
| **규격/표준** | KS/KDS/EN/ASTM/ACI/BS 규격 13건 |

## 2. AI 질의응답 사용법

### 질문 예시
- "SG 80ES 압축강도 알려줘"
- "SPPM 400 시공방법은?"
- "SG 70S와 Fosroc Conbextra 비교"
- "KS F 4044 기준값 전부 알려줘"
- "콘크리트 탄산화란?"

### 답변 구성
- **마크다운 렌더링**: 굵은 글씨, 목록, 테이블 자동 포맷팅
- **출처 표시**: 답변 하단 출처 버튼 클릭 → 오른쪽 상세 패널

## 3. PDF 업로드
1. 좌측 메뉴 **"업로드"** 클릭
2. PDF 파일 드래그앤드롭 또는 파일 선택
3. 자동 분류 → 텍스트 추출 → AI 임베딩 생성
4. AI 채팅에서 해당 PDF 내용으로 질문 시 출처로 표시

## 4. 위키 문서
- **작성**: 위키 목록 → "새 문서" → Markdown 에디터
- **편집**: 위키 상세 → "편집" 버튼 → 버전 자동 증가

## 5. 설정
- **테마**: 헤더 우측 Sun/Monitor/Moon (라이트/시스템/다크)
- **폰트**: 헤더 우측 -/보통/+ (13px/14px/16px)
