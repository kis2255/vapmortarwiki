## 2. PDF 자료 업로드

### 업로드 방법
1. 좌측 메뉴에서 **"PDF 관리"** 클릭 또는 상단 **"업로드"** 버튼 클릭
2. PDF 파일을 **드래그앤드롭**하거나 **"파일 선택"** 클릭
3. 여러 파일 동시 업로드 가능

### AI 자동 분류
시스템이 PDF 내용을 분석하여 문서 유형을 자동 판별합니다:

| 유형 | 설명 | 파일명 팁 |
|------|------|----------|
| TDS | 기술자료 (배합비, 시공법) | 파일명에 `TDS` 포함 |
| MSDS | 안전보건자료 | 파일명에 `MSDS` 또는 `SDS` 포함 |
| 시험성적서 | KS 시험 결과 | 파일명에 `시험` 또는 `test` 포함 |
| 인증서 | KS인증, 환경표지 등 | 파일명에 `인증` 또는 `cert` 포함 |
| 시공사례 | 현장 적용 사례 | 파일명에 `시공` 또는 `case` 포함 |
| 카탈로그 | 제품 안내 자료 | 파일명에 `카탈` 또는 `catalog` 포함 |
| 기술논문 | 연구/학술 자료 | - |

> **팁**: 파일명에 문서유형을 명시하면 분류 정확도가 높아집니다.
> 예: `TDS_RM-100_v3.2.pdf`, `시험성적서_RM-200_KICT.pdf`

### 분류 확인 및 수정
- AI가 분류한 결과를 확인하고, 필요시 드롭다운에서 올바른 유형을 선택
- 제품과 연결: 해당 자료가 속하는 제품을 선택
- **"업로드 완료"** 버튼으로 확정

## 3. 제품 등록

### 새 제품 등록 순서
1. 좌측 메뉴 **"제품 DB"** → **"제품 등록"** 클릭
2. 기본 정보 입력:
   - **제품코드** (필수): `RM-100`, `WP-200` 형식
   - **제품명** (필수): `VAP 보수몰탈 RM-100`
   - **카테고리** (필수): 보수몰탈 / 방수몰탈 / 바닥몰탈 / 주입재 / 그라우트
3. 시공 정보 입력:
   - 적용범위, 시공방법, 양생조건
4. 물성 데이터 입력:
   - **"항목 추가"** 버튼으로 행 추가
   - 시험항목명, 단위, 규격기준, 시험결과, 시험방법 입력

### 물성 데이터 입력 예시
| 시험항목 | 단위 | 규격기준 | 시험결과 | 시험방법 |
|---------|------|---------|---------|---------|
| 압축강도(28일) | MPa | ≥40 | 52.3 | KS F 2476 |
| 휨강도(28일) | MPa | ≥7 | 9.8 | KS F 2476 |
| 부착강도 | MPa | ≥1.5 | 2.1 | KS F 2476 |

## 4. 위키 문서 작성

### 문서 작성 순서
1. 좌측 메뉴 **"위키 문서"** → **"문서 작성"** 클릭
2. 문서 제목, 카테고리, 태그 입력
3. **Markdown** 형식으로 본문 작성
4. **"미리보기"** 버튼으로 결과 확인
5. **"문서 발행"** 클릭

### Markdown 작성 가이드

```markdown
# 대제목

## 중제목

### 소제목

일반 텍스트입니다. **굵은 글씨**를 사용할 수 있습니다.

- 목록 항목 1
- 목록 항목 2

| 항목 | 값 | 단위 |
|------|-----|------|
| 압축강도 | 52.3 | MPa |
| 휨강도 | 9.8 | MPa |
```

### 추천 문서 주제
- 제품별 기술 해설 (PCM이란?, 무수축 그라우트란?)
- KS 규격 해설 (KS F 4042, KS F 4044 등)
- 시공 가이드 (단면보수 순서, 균열주입 방법)
- 시장 동향 (국내/글로벌)
- FAQ (자주 묻는 기술 질문)

## 5. AI 질의응답 활용

### 사용 방법
1. 좌측 메뉴 **"AI 질의응답"** 클릭 또는 상단 **"AI 질의"** 버튼
2. 자연어로 질문 입력
3. AI가 등록된 자료를 검색하여 답변 + 출처 표시

### 질문 예시
| 질문 유형 | 예시 |
|----------|------|
| 제품 물성 | "RM-100의 압축강도는?" |
| 제품 비교 | "RM-100과 RM-200의 차이점" |
| 규격 확인 | "KS F 4042 기준값은?" |
| 시공 방법 | "보수몰탈 시공 순서 알려줘" |
| 제품 추천 | "방수에 적합한 제품 추천해줘" |
| 시장 정보 | "국내 보수보강 시장 전망은?" |

### 주의사항
- AI는 **등록된 자료만 기반**으로 답변합니다
- 자료가 없으면 "확인된 자료가 없습니다"로 답변합니다
- 물성 수치는 DB에서 직접 조회하므로 정확합니다
- 답변 하단의 **출처**를 클릭하면 원본 문서를 확인할 수 있습니다

## 6. 통합 검색

1. 상단 검색바에 키워드 입력 후 Enter
2. 결과를 유형별로 필터링: 전체 / 제품 / 위키문서 / PDF문서 / 규격
3. 검색 예시:
   - 제품코드: `RM-100`
   - 물성: `압축강도 40MPa`
   - 규격: `KS F 4042`
   - 키워드: `방수 지하`

## 7. 자료 입력 우선순위 (권장)

### 1단계: 제품 기본 등록
- [ ] 전체 제품 코드/이름/카테고리 등록
- [ ] 주요 물성 데이터 입력

### 2단계: 핵심 PDF 업로드
- [ ] 각 제품별 TDS (기술자료)
- [ ] 각 제품별 MSDS (안전보건자료)
- [ ] 최신 시험성적서

### 3단계: 위키 문서 작성
- [ ] 제품 카테고리별 기술 해설
- [ ] KS 규격 해설
- [ ] 시공 가이드

### 4단계: 보충 자료
- [ ] 인증서
- [ ] 시공사례 (사진 포함)
- [ ] 카탈로그/브로슈어
