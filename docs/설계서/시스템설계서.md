## 1. 시스템 개요

### 1.1 목적
삼표산업 특수몰탈 사업부의 기술자료 데이터베이스 및 AI 기반 질의응답 시스템. 제품 물성, 시공방법, KS/EN 규격, 시장 동향, 글로벌 경쟁사 분석을 통합 관리한다.

### 1.2 주요 기능
| 기능 | 설명 |
|------|------|
| 제품 DB | 삼표 15종 + 경쟁사 8종 제품/물성/규격 관리 |
| 위키 문서 | 기술자료 마크다운 문서 작성/편집/검색 |
| AI 채팅 | RAG 기반 기술 질의응답 (Gemini 2.0 Flash) |
| PDF 관리 | PDF 업로드 → 텍스트 추출 → 벡터 임베딩 → AI 검색 |
| 통합 검색 | 제품/위키/PDF/규격 통합 검색 |
| 프로젝트 현황 | Asana 연동 PI 프로젝트 대시보드 (간트 차트/테이블) |

## 5. RAG 파이프라인

```
[사용자 질문]
    ↓
[의도 분류] (Gemini Flash)
    ├── LOOKUP    → 제품 코드 직접 조회
    ├── COMPARE   → SQL + RAG 검색
    ├── RECOMMEND → SQL 필터 + RAG 검색
    ├── EXPLAIN   → RAG 검색
    └── GENERAL   → RAG 검색
    ↓
[하이브리드 검색]
    ├── pgvector 코사인 유사도 (vector <=> query_vector)
    ├── 다중 키워드 ILIKE OR (한국어 조사 제거, 최대 5개)
    └── 제품 DB 직접 조회 (모든 의도에서 실행)
    ↓
[컨텍스트 구성] (검색 결과 + 제품 데이터)
    ↓
[답변 생성] (Gemini 2.0 Flash + 시스템 프롬프트)
    ↓
[응답] (마크다운 답변 + 출처 링크)
```

### 5.1 임베딩 사양
- 모델: `gemini-embedding-001`
- 차원: 768
- 인덱스: pgvector HNSW
- 검색: 코사인 유사도 (`1 - (vector <=> query_vector)`)

### 5.2 키워드 검색
- 한국어 어미/조사를 단어 경계에서만 제거
- 다중 키워드 ILIKE OR 쿼리
- 키워드 매칭 가중치(0.8) > 벡터 유사도 가중치(0.5)

## 9. 배포

### 9.1 배포 구성

| 항목 | 내용 |
|------|------|
| 서버 | 10.50.20.51 (Azure VM, Ubuntu 24) |
| 도메인 | vapm.sampyo.co.kr |
| 포트 | 8003 (호스트) → 3000 (컨테이너) |
| 배포 스크립트 | `python scripts/deploy_PROD.py` |
| Docker | app (Next.js standalone) + DB (pgvector:pg17) |
| Nginx | HTTPS 리버스 프록시 + HTTP→HTTPS 리다이렉트 |

### 9.2 배포 절차

```bash
# 1. 로컬에서 push
git push origin main

# 2. 배포 실행
GH_TOKEN=ghp_xxx python scripts/deploy_PROD.py
```

스크립트 자동 실행 순서:
1. SSH 보호
2. 패키지 확인 (git, nginx, docker)
3. git pull + .env.prod 업로드
4. Docker 빌드 + 실행 (Next.js 빌드 포함)
5. Nginx HTTPS 설정
6. DB 스키마 + pgvector 확인
7. 헬스체크

상세: `docs/배포기준서/배포절차.md` 참조
