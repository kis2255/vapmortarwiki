## 7. 상태 표시

### 7.1 프로젝트 신호등

| 아이콘 | 의미 | 조건 |
|--------|------|------|
| 🟢 | 순조 | Asana green 또는 진행률 ≥50% |
| 🔵 | 진행중 | Asana blue 또는 기본 상태 |
| 🔴 | 위험 | Asana red 또는 기한초과/30일 이내+진행률<50% |
| ⚪ | 미설정 | 상태 미등록 |

### 7.2 제품 물성 합격/불합격

| 표시 | 조건 |
|------|------|
| ✅ 합격 (녹색) | 실측값 ≥ 기준값 |
| ❌ 불합격 (빨강) | 실측값 < 기준값 |
